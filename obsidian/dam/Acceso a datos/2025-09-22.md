# Conversiones con tipos de referencia
## Downcasting/Reducción
```java
class Padre {
	mSoloEnPadre() {}
	mEnAmbos()
	{
		1
	}
}
class Hija extends Padre {
	mSoloEnHija() {}
	@Override
	mEnAmbos()
	{
		2
	}
}

Padre p = new Hija();
p.mEnAmbos() // 2, si comparten método, se llama al de la subclase

((Hija)p).mSoloEnHija() // p es Hija en memoria, por eso puede acceder a los métodos de dicha clase 
```

```java
class Padre {}
class Hijo extends Padre implements IInterfaz {}
interface IInterfaz {}

Padre obj1 = new Padre();
Padre obj2 = new Hijo();

obj1 instanceof Padre // true
obj1 instanceof Hijo // false
obj1 instanceof IInterfaz // false, la clase Padre no implementa la interfaz

obj2 instanceof Padre // true
obj2 instanceof Hijo // true
obj2 instanceof IInterfaz // true 

Child obj3 = new Child();
obj3 instanceof IInterfaz // true
```
